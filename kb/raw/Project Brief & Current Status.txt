## 🏗️ ARCHITECTURE DESIGN

### Configuration Philosophy
**Clean Separation Principle:**
- **`configuration.nix`**: System-level stability (OS, drivers, services, GUI apps)
- **`flake.nix`** (system): NixOS system wrapper
- **`llm-agentic-lab-flake.nix`**: Development environments (CUDA, ML stack, Python)

### Key Design Decisions
1. **NO `cudaSupport = true` globally** → Prevents system-wide CUDA pollution
2. **NO ML packages in system config** → Keeps rebuilds fast
3. **PRIME offload over sync** → Better power management, explicit GPU usage
4. **Specialisation for battery** → Boot option for portable use
5. **Wayland-first** → Modern display protocol, better Electron app support
